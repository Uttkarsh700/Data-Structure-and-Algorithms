📘 Data Structures and Algorithms (DSA) Practice in C++

A curated collection of C++ solutions to fundamental Data Structures and Algorithms problems.
This repository is designed for consistent practice and strengthening core programming concepts.

📂 Repository Structure
🧠 Practice Problems
binomial_coefficient.cpp → Computes binomial coefficients (nCr)
char_occur.cpp → Finds the next character in the alphabet sequence
even_odd.cpp → Checks whether a number is even or odd
factorial.cpp → Calculates factorial of a number
function1.cpp → Basic function implementation practice
function2.cpp → Additional function-based exercises
inrange_prime.cpp → Finds prime numbers within a given range
is_palindrome.cpp → Checks whether a number/string is a palindrome
is_prime.cpp → Checks whether a number is prime
is_prime_optimized.cpp → Optimized prime number checking algorithm
largest_of_three_numbers.cpp → Finds the largest among three numbers
sum_of_digits.cpp → Calculates the sum of digits of a number
