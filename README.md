# Data Structure and Algorithms Practice

A collection of DSA practice problems solved in C++. This repository contains solutions to various fundamental programming problems and algorithms.

## Contents

### Practice Questions
- **bionomial_cofficent.cpp** - Calculate binomial coefficients
- **charoccur.cpp** - Find the next character in the alphabet
- **even_odd.cpp** - Determine if a number is even or odd
- **factorial.cpp** - Calculate factorial of a number
- **function1.cpp** - Basic function practice
- **function2.cpp** - Additional function exercises
- **inrangeprime.cpp** - Find prime numbers in a given range
- **ispallindrome.cpp** - Check if a number/string is a palindrome
- **isprime.cpp** - Check if a number is prime
- **isprime_optimized.cpp** - Optimized prime checking algorithm
- **largestofthreenum.cpp** - Find the largest of three numbers
- **sumofdigits.cpp** - Calculate the sum of digits

## Getting Started

### Prerequisites
- C++ compiler (g++, clang, or MSVC)

### Compilation
```bash
g++ filename.cpp -o output
./output
```

- Basic Programming Concepts

- Mathematical Algorithms
